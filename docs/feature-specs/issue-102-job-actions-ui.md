# 이슈 102. Job action UI

## 목적

Job 취소, 실패 항목 재시도, 결과 다운로드 같은 사용자 action을 화면에서 수행할 수 있게 합니다.

## 작업 범위

1. Job 취소 버튼
2. 실패 항목 재시도 버튼
3. 전체 결과 다운로드 버튼
4. action 실행 중 상태 표시
5. 성공/실패 toast 또는 message 표시

## 완료 기준

1. 완료된 Job에는 불가능한 action을 숨기거나 비활성화합니다.
2. action 실패 이유를 사용자에게 보여줍니다.
3. API 상태와 화면 상태가 어긋나지 않습니다.
