# 이슈 87. ZIP 업로드와 결과 ZIP 다운로드

## 목적

사용자가 ZIP 파일을 업로드하면 이미지 파일을 추출해 전처리하고, 처리된 이미지만 ZIP으로 다시 다운로드할 수 있게 합니다.

## 작업 범위

1. ZIP 파일 선택 지원
2. ZIP 내부 이미지 추출
3. 지원하지 않는 파일 무시 또는 실패 처리
4. 이미지별 upload/session 등록
5. Job 생성
6. processed-only ZIP 다운로드

## 완료 기준

1. ZIP 결과에는 처리된 이미지 중심으로 포함합니다.
2. preview/report는 사용자 기본 다운로드에 포함하지 않습니다.
3. 대량 파일 처리 시 파일별 실패 상태를 확인할 수 있습니다.
