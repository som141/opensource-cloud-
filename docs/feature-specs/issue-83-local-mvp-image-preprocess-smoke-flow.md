# 이슈 83. 로컬 MVP 이미지 전처리 스모크 플로우

## 목적

브라우저에서 로그인, 업로드, Job 생성, Worker 처리, 결과 확인까지 한 번에 테스트할 수 있는 로컬 흐름을 제공합니다.

## 작업 범위

1. 로그인 후 upload 화면 이동
2. 프로젝트 생성 또는 선택
3. 이미지 업로드
4. 전처리 Job 생성
5. 진행률 확인
6. 처리 결과 다운로드

## 완료 기준

1. Docker Compose local 스택에서 동작합니다.
2. 실제 Worker가 처리한 결과를 확인할 수 있습니다.
3. 실패 시 화면에 명확한 오류 메시지를 보여줍니다.
